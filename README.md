# 🟢 Simon Says Game

A fun and colorful **Simon Says** memory game using HTML, CSS, and JavaScript.

## 🧠 How to Play
- Watch the color pattern and repeat it.
- Each level adds a new color.
- One mistake and it's game over!

## 🚀 Features
- Progressive levels
- Fully responsive

## 💻 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
